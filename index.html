Ezra

自動經文顯示

View on GitHub Download .zip Download .tar.gz

Ezra是中文的Reftagger。Ezra能夠找出網頁中的所有聖經依據，如:約 1:2;弗 2:1，然後顯示經文。當光標移到經文依據時，Ezra就會顯示相應的經文。尤其適合信仰告白，因為每句都是充滿一串串的經文依據。

特別鳴謝「信望愛信仰與聖經資源中心」提供了聖經JSON API。

本服務正進行封閉測試，有興趣使用的人士請與我聯絡。
